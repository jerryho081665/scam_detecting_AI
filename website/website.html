全面防護

強大的六大核心功能

不只是錄音，我們提供全方位的 AI 防詐解決方案。

雙引擎語音轉錄

整合 Google 原生語音識別與台灣在地化「雅婷逐字稿」(Yating)，精準識別中文與台語混雜的對話內容。

即時風險偵測

搭載本地輕量化 BERT 模型，以毫秒級速度計算詐騙機率。一旦偵測到高風險關鍵字，立即觸發紅色警示。

生成式 AI 建議

連接 GPT-4o 或 Qwen 大語言模型，不僅告訴你有危險，還能分析詐騙邏輯並教你如何反擊或掛斷。

語音警示廣播

專為長者設計。當風險極高時，APP 會透過語音大聲朗讀：「注意！偵測到高風險詐騙內容」，即時打斷對話。

歷史紀錄存證

自動將可疑通話的文字稿存入本地加密資料庫 (Room DB)。您可以隨時查看、編輯或匯出紀錄作為報案證據。

開發者自訂模式

針對進階用戶，開放自訂 API 端點與 JSON 模板。您可以切換不同的 AI 模型或連接至私有伺服器，彈性極高。

雙層防護架構：快慢結合

我們獨創的「快慢雙層架構」解決了即時性與準確性的矛盾。確保在通話進行時，能同時獲得毫秒級的反應與深度的 AI 分析。
